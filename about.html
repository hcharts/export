How to use ?

1. Create a code

Go http://code.hcharts.cn to begin a new code, you can type Javascript, Html, Css code as you like, you can choose js lib for highcharts need in left silder.

There are some point you should rember

  • The Html and Javascript code is required;
  • You can add other js lib in html editor after your html main code.See the Demo

  • The Javscript editor only accept javascript code, you should't type any other code such as "<script type="text/javascript">", this code will result in errors.

2. Run

When you are ready to see what is your code look like, you can click "Run" to run and preview your code in "Result" box.

3. Save or update

Before save your code ,you can Named your code and give a short text to description your code.

Before saved, you should specific your domain, such "admin", then your special domain is "http://code.hcharts.cn/admin". it is unchangeable after setting.

The last thing is click "Update" to save or update your code. when you are editing other people's code , save means fork.

4. Share

You can show your code by sharing url, full with result and embed on your page.

embed options :

http://code.hcharts.cn/{your_domain}/{code}/{tables}

{tables} can be "result", "js", "html", "css" , two or more tables was separated by ",".

Eg :

  • http://code.hcharts.cn/highcharts/hhhhhm/result
  • http://code.hcharts.cn/highcharts/hhhhhm/result,js,html
  • http://code.hcharts.cn/highcharts/hhhhhm/result,css,js,html
  • http://code.hcharts.cn/highcharts/hhhhhm/css,js,html,result
  • ..

Demo :

Demo code : 

<iframe width="100%" height="400" src="http://code.hcharts.cn/highcharts/hhhhhn/result,js,html" allowfullscreen="allowfullscreen" frameborder="0"></iframe>
