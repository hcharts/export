Exporting

Highcharts 导出服务

轻松将你的 Highcharts 图表导出下载为图片或 PDF 文档

Learn more

使用方式

1、highcharts导出服务器 

$('#container').highcharts({
    // ... Highcharts 配置

    exporting: {
        url: 'http://export.hcharts.cn/'
    }

    // ... Highcharts 配置
});

2、csv 导出服务器

Highcharts.post('http://export.hcharts.cn/csv.php', {
    csv: chart.getCSV()
});

搭建本地服务器

以上的所有导出服务器都可以在本地搭建，目前我们已经提供 php、java、asp、nodejs 等不同版本的导出服务器及详细的文档，详见文档。
